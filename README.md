# Análise Estruturada de Discursos Parlamentares Brasileiros

## Objetivo
Demonstrar um pipeline de coleta, limpeza e análise de discursos parlamentares utilizando dados públicos.

## Arquitetura
1. Coleta de dados públicos
2. Limpeza e normalização
3. Classificação ideológica
4. Agregações temporais
5. Exportação para visualização interativa

## Stack
Python, Pandas, JSON, HTML/JS

## Observação
Este repositório contém uma versão demonstrativa da pipeline.

   ↓
